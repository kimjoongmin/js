    5장
    -함수

      변수 - 데이터 저장공간, 선언할때 var 사용
      함수 - 스크립트를 저장하는 공간, 선언할때 function 을 사용

      *기본형
      function 함수명 (){
        스크립트 코드
      }
      함수명();

      *익명함수
      참조변수 = function(){
        스크립트 코드
      }
      참조변수();

      *매개변수
      function 함수명(매개변수1, 매개변수2){
        스크립트 코드;
      }
      함수명(값1,값2);
    

    -스코프
      스코프란? 변수 또는 함수의 이용 가능한 유효 범위
      지역변수? 스코프 영역에서 var를 붙여 선언된 변수를 가리키며, 스코프 영역내에서만 이용 가능
      전역변수? 함수 밖에 선언된 변수
      내부함수? 스코프 영역에서 선언된 함수를 가리키며, 스코프 영역 내에서만 호출
