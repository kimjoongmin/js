    6장
    -제이쿼리 : 자바스크립트 라이브러리 언어
                jquery.com/download 
                cdnjs.com

    -선택자 : 선택자는 html 요소를 선택하여 가져옴
      $(function(){
        $("선택자").css("속성","값");
      })

      -기본 선택자 : 직접선택자, 간접선택자
      -탐색 선택자 : 위치선택자, 속성선택자
      -객체 조작 : 선택한 요소를 삭제, 생성, 복제, 속성변경

    -DOM(Document Object Model) : html 문서 객체 구조를 말합니다


    7장
    -이벤트 : 사이트에서 방문자가 취하는 모든 동작
    -핸들러 : 이벤트가 발생했을 때 호출되는 함수
    
      -이벤트 대상 : 버튼1,버튼2
      -이벤트 종류 : click, mouseover
      -이벤트 핸들러 : 경고창

    

    8장
    -효과 및 애니메이션

    -효과 메서드
      $(요소선택).효과 메서드(효과 소요 시간, 가속도, 콜백함수);
    -애니메이션 사용법
      $(요소선택).animate({애니메이션에 적용할 속성과 값}, 가속도, 콜백함수);
