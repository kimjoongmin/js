    7.논리 연산자 (AND, OR, NOT)
    or || 여러개중 하나라도 true면 true 즉 모든값이 false 일때만 false를 반환
    and && 여러개중 모두 true여야 true 즉 하나라도 false가 있으면 false를 반환
    not ! true면 true false면 false

    or는 첫번쨰 true를 발견하는 즉시 평가를 멈춤
    and는 첫번째 false를 발견즉시 멈춤
    
    ex)운전면허(80%)가 있고 시력(60%)이 좋은 여군(7%)
    프로그램 짤때
    여군이고 시력이 좋고 운전면허가 있는 사람으로 짜는게 좋음
    첫 여군에서 93%를 걸러낼수 있음

    ***객체프로퍼티로 할당 된 함수 = method 

    const superman = {
      name : 'clark',
      age : 30,
      fly : function(){ //fly(){줄여쓸수있음
        console.log('날아갑니다')
      },
      sayHello : function (){
        console.log(`Hello, ${this.name}`)
      }
    }
    superman.sayHello();

    object
    화살표 함수는 일반 함수와는 달리 자신만의 this를 가지지 않음
    화살표 함수 내부에서 this를 사용하면, 그 this는 외부에서 값을 가져옴
    let boy = {
      name : 'Mike',
      age : 30,
      sayHello : () => {
        console.log(this) //전역객체
      }
    }
    boy.sayHello();
    this != boy

    14.배열(Array)
    순서가 있는 리스트

    **배열의 특징
    배열은 문자 뿐만아이니라 숫자 객체 함수 포함
    length 배열의 길이
    push(); 배열끝 요소추가
    pop(); 배열끝 요소 제거
    unshift(); 배열 처음에 추가 여러요소 한꺼번에 추가가능
    shift(); 배열 처음 제거

    배열사용하는 가장 큰이요 반복을위해 
    for문 사용
    for of문 사용(객체는 for in)
    for(let day of days){
      console.log(day)
    }
    for문 보다는 간단하지만 index를 얻을 수 없음.
