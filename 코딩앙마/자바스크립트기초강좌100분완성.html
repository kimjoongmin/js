    4.형변환
    -String() - 문자형으로 변환
    -숫자형으로 변환 - Number() 
    true=1, false=0
    Number("문자") // NaN
    Number(null) // 0
    Number(undefined) // NaN
    -불린형으로 변환 - Boolean()
    false = 숫자0, 빈문자열'', NaN, null, undefined 
    true = 위 제외 모두
    형변환 필요 - 의도치않은 동작 방지(명시적형변환)
    주의사항
    Number(0) // false
    Number('0') //true
    Number('') //false
    Number(' ')//true

    7.논리 연산자 (AND, OR, NOT)
    or || 여러개중 하나라도 true면 true 즉 모든값이 false 일때만 false를 반환
    and && 여러개중 모두 true여야 true 즉 하나라도 false가 있으면 false를 반환
    not ! true면 true false면 false

    or는 첫번쨰 true를 발견하는 즉시 평가를 멈춤
    and는 첫번째 false를 발견즉시 멈춤
    
    ex)운전면허(80%)가 있고 시력(60%)이 좋은 여군(7%)
    프로그램 짤때
    여군이고 시력이 좋고 운전면허가 있는 사람으로 짜는게 좋음
    첫 여군에서 93%를 걸러낼수 있음



    8.반복문(for, while, do while)
    for(let i=0;i < 10;i++){
      반복할 코드
    }
    let i = 0;
    while(i < 10){
      //코드
      console.log(i)
      i++;
    }
    let i =0;
    do{
      //코드
      i++;
    }while(i < 10)
    do while은 적어도 한번은 실행 

    break; 멈추고 빠져나옴
    continue  멈추고 다음 반복으로 진행

    명확한 횟수가 정해져 있으면 for문을 
    그렇지 않으면 while문을 사용
