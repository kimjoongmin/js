    var : 함수 스코프
    한번 선언된 변수를 다시 선언가능
    선언하기전에 가능
    호이스팅!!! 
    선언은 호이스팅 되지만 할당은 호이스팅 안됨
    var name;
    console.log(name); //undefined
    name = 'Mike'; //할당

    *호이스팅 : 스코프 내부 어디서든 변수 선언은 최상위에 선언된 것 처럼 행동
    호이스팅은 스코프 단위로 일어남
    
    let과 const는 tdz때문에 안됨 temporal dead zone

    2.초기화 단계
    3.할당 단계

    1.선언 및 초기화단계(undefined를 할당 해주는 단계)
    2.할당 단계

    let
    1.선언단계
    2.초기화단계
    3.할당단계

    const 1.선언 +초기화+할당
    var name;
    name = 'Mike';
