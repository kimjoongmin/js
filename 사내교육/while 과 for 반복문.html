while 과 for 반복문

반복문 loop 을 사용하면 동일한 코드를 여러번 반복가능

while 반복문
while (condition) {
    //코드
    //반복문 본문 (body)라 불림
}
condition(조건)에 truthy이면 반복문 본문의 코드가 실행

아래 반복문은 조건 i < 3 을 만족할 동안 i를 출력

let i = 0;
while(i < 3){
   alert(i)
   i++; 
}
반복문이 한번 실행되는것을 이터레이션
위예시에선 반복문이 세번의 이터레이션을 만듬
i++가 없었다면 이론적으로 반복문이 영원히 반복되었을 겁니다

그런데 브라우저는 이런 무한 반복을 멈추게하는 실질적인 수단을 제공합니다
서버 사이드 자바스크립트도 이런 수단을 제공해 주므로 무한으로 반복되는 프로세스를 종료할수 있다

let sum = 0;

while(true){

    if(!value) breack;//*

}

*표시 한 줄에 있는 break는 사용자가 아무것도 입력하지 않거나 cancel버튼을 눌렀을때 활성화
이때 반복문이 즉시 중단되고 제어 흐름이 반복문 아래 첫번째줄로 이동.
여기선 alert가 그 첫번쨰 줄

반복문의 시작지점이나 끝 지점에서 조건을 확인하는 것이 아니라 본문 가운데 혹은 본문 여러 곳에서 조건을 확인해야 하는 경우, 
무한반복문 + break

다음 반복으로 넘어가기
continue 지시자는 break의 가벼운 버전
continue 는 전체 반복문을 멈추지 않습니다
대신에 현재 실행중인 이터레이션을 멈추고 반복문이 다음 이터레이션을 강제로 실행(조건통과시)

continue는 현재 반복을 종료시키고 다음 반복으로 넘어가고 싶을때 사용

for(let i = 0;i < 10;i++){
    if(i % 2 == 0) continue; //조건이 참이면 남아있는 본문은 실행이 안됨
    
    alert(i); // 1, 3, 5, 7 ,9 출력
}
i가 짝수이면 continue가 본문 실행을 중단시키고 다음 이터레이션이 실행되게합니다
(i가 하나 증가하고, 다음 반복이 실행됨.) 따라서 alert 함수는 인수가 홀수일때만 호출
continue는 중첩을 줄이는데 도움
홀수 출력 예시 
for(let i = 0; i < 10; i++){
    if(i % 2){
        alert(i);
    }
}
기술적으로 동일하지만 부작용으로 중첩 레벨(대괄호안의 alert호출)이 하나 더 늘어남
if안의 코드가 길어지면 전체 가독성 떨어짐

표현식이 아닌 문법구조는 삼항 연산자 ? 에 사용할수 없다
특히 break 이나 continue 같은 지시자는 삼항 연산자에 사용하면 안됨

if(i < 5){
    alert(i);
} else {
    continue;
}
(i < 5) ? alert(i) : continue; //사용불가

break/continue와 레이블
여러개의 중첩 반복문을 한번에 빠져나와야하는 경우 종종 있음
i와 j를 반복하면서 프롬프트창에 (0,0) 부터 (2,2)까지를 구성하는 좌표 (i,j)를 입력하는 예시
for(let i =0; i < 3; i++){
    for(let j = 0; j < 3; j++){
        let input = prompt(`${i},${j}의 값`,'');
        //여기서 멈춰서 아래쪽의 "완료!" 가 출력되게 하려면 어떻게?
    }
}
alert("완료!");
사용자가 cancel 버튼을 눌렀을때 반복문을 중단시킬 방법 필요
input 아래에 평범한 break 지시자를 사용하면 안쪽에 있는 반복문만 빠져나올수 있음
이것만으론 충분치 않음(중첩반복문을 포함한 반복문 두개 모두를 빠져놔와야하기때문)
이럴때 레이블 사용
레이블(label)은 반복문 앞에 콜론과 함께쓰이는 식별자

labelName : for(...){
    ...
}
반복문 안에서 break < labelNmae >을 사용하면 레이블 해당하는 반복문 빠져나올수 있음

outer : for(let i = 0; i < 3; i++){
    for(let j = 0; j < 3; j++){
        let input = prompt(`${i},${j}의 값`,"");

        //사용자가 아무것도 입력하지 않거나 cancel버튼을 누르면 두 반복문을 빠져나옴
        if(!input) break outer; //*

        //입력받은 값을 가지고 무언가를 함
    }
}
alert("완료!");
위 예시에서 break outer 는 outer 라는 레이블이 붙은 반복문을 찾고, 해당 반복문을 빠져나오게해줌
따라서 제어흐름이*에서 alert("완료!")로 바로 바뀜
