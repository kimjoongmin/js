2.1표현식 구문
    구문구별~ 표현식은 값을 가짐
    6*7 은 값이 42인 표현식
    Math.max(6,7) 은 메서드 호출도 표현식
    구문은 값을 갖지 않는 대신 기능을 포함

    2.2세미콜론 자동 추가 규칙
    자바스크립트는 자동으로 세미콜론 추가

    2.3분기
    if(조건) 구문
    조건부 연산자 ? : 를 사용
    let max = undefined;
    if(x > y) max = x; else max = y;
    let max = x > y ? x : y;

    2.4 불리언화
    자바스크립트의 조건은 불리언 값을 사용
    0, NaN, null, undefined, 빈 문자열 은 조건을 거짓으로 만드는 일종의 거짓 값
    이를 제외한 모든 다른값은 일종의 참

    2.5 비교연산자
    3 < 4 true 
    3 >= 4 false
    NaN < 4  false
    NaN >= 4 false 
    NaN <= NaN false
